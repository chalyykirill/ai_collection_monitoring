# Playbook качества модели

При падении Gini:

- сравнить target rate по периодам и сегментам;
- проверить PSI score и PSI признаков;
- сравнить распределения score;
- проверить missing rate и пустые feature vector;
- проверить продуктовый, DPD и сегментный состав потока.

Без этих проверок root cause не считается подтвержденным.
